# [PRD] 1인 가방 브랜드 전용 멀티 채널 상세페이지 제작 솔루션

## 1. 왜 만드나 (문제와 목표)
* **문제:** 6개(자사몰, 29CM, 무신사, W컨셉, 스마트스토어, 쿠팡)의 서로 다른 채널을 혼자 운영하며, 채널별로 요구하는 상세페이지 사이즈와 문구, 구성 규칙을 매번 수동으로 맞추느라 과도한 시간이 소요됨.
* **목표:** 제품 사진과 브랜드 자산만 입력하면, 6개 채널별 규격에 맞춘 상세페이지와 대표 이미지(썸네일)를 자동으로 생성·다운로드하여 **콘텐츠 제작 시간을 획기적으로 단축**하고 운영 효율성을 높임.

## 2. 누가 쓰나 (사용자와 상황)
* **사용자:** 자체 제작 가방 브랜드 'immm'을 운영하는 1인 기업 대표.
* **상황:** 이동 중이거나 외부에서 스마트폰을 사용하여 틈틈이 업무를 처리해야 함. 복잡한 디자인 툴(포토샵 등)을 켜지 않고, **최소한의 클릭으로 모바일에서 즉시 결과물을 얻길 원함.**

## 3. 무엇을 하나 (핵심 기능)

### A. 스마트 썸네일 생성
* **1000px * 1000px:** 쿠팡, 스마트스토어, 29CM, W컨셉
* **1500px * 1500px:** 무신사
* **로직:** 원본 이미지를 업로드하면 위 규격에 맞춰 자동 리사이징 또는 패딩 처리하여 1:1 정방형으로 생성.

### B. 채널별 상세페이지 자동 리사이징 및 분할
* **규격 가이드:**
    * **네이버 스마트스토어:** 가로 860px (세로 최대 5,000px 권장)
    * **쿠팡:** 가로 780px (세로 최대 30,000px)
    * **29CM:** 가로 750~1000px (**세로 8,000px 초과 시 자동 분할**)
    * **무신사:** 가로 1500px / 세로 1500px (정방형 고정)
    * **W컨셉:** 가로 860px (세로 최대 5,000px)
* **로직:** 채널 선택 시 가로 폭에 맞춰 비율 유지 리사이징을 수행하며, 29CM 등 특수 조건(자동 분할 등)을 충족하도록 처리.

### C. 브랜드 자산 및 템플릿 관리
* **기능:** 채널별 할인/배송 안내 문구 프리셋 설정.
* **기능:** 신제품 제작 시 로고, 소재 정보 등 브랜드 자산을 클릭 한 번으로 불러오기.

### D. 일괄 다운로드
* **기능:** 6개 채널별 결과물을 구분하여 ZIP 파일로 일괄 압축하거나, 채널별로 즉시 다운로드 가능한 모바일 UI 제공.

## 4. 무엇은 안 하나 (이번 버전에서 뺄 것)
* **외부 채널 API 직접 연동:** 채널 관리자 페이지로 직접 업로드하는 기능은 제외(파일 생성 및 다운로드까지만 구현).
* **고도화된 레이어 편집:** 전문 디자인 툴 수준의 필터, 자유로운 레이어 편집은 제외(템플릿 기반 자동화에 집중).
* **협업 및 분석:** 사용자 간 공유, 팀 협업, 판매 데이터 통계 기능은 제외.
